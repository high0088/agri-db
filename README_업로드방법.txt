GitHub 업로드 방법

1. 이 압축파일을 풀기
2. GitHub 저장소 agri-db의 Code 화면에서 uploading an existing file 클릭
3. 아래 6개 파일을 모두 업로드
   - index.html
   - manifest.json
   - service-worker.js
   - icon-192.png
   - icon-512.png
   - apple-touch-icon.png
4. Commit changes 클릭
5. Settings > Pages > Source: Deploy from a branch > Branch: main > Save
6. 주소: https://high0088.github.io/agri-db/

주의: 입력 데이터는 각 기기의 브라우저 저장소에 저장됩니다. 다른 PC/아이폰으로 옮길 때는 프로그램 안의 전체 데이터 저장(JSON) 기능을 사용하세요.
